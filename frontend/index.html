features

核心特性

运行时与安全基座

kernel + ssl + utils 提供协程调度、异步 IO、TLS 安全传输与通用组件，作为全套库统一基座。

协程调度 异步 IO 事件循环 kqueue epoll io_uring TLS 1.2/1.3

协议与服务通信

覆盖 HTTP/1.1、HTTP/2、WebSocket 与 RPC 能力，支持路由、静态文件、连接升级与服务间调用。

HTTP/1.1 HTTP/2 WebSocket RPC Route / Static

数据客户端能力

提供 Redis、MySQL、Mongo、etcd 客户端，支持同步/异步调用、认证、事务与批量管线操作。

Redis MySQL MongoDB etcd Sync / Async Pipeline

客户端/服务端同构

同一套编程模型可覆盖客户端与服务端，支持同步/异步并存，便于跨项目复用和演进。

Client Server Sync / Async macOS / Linux

AI 协议接入

支持 MCP 协议（stdio/HTTP、JSON-RPC），用于工具注册与 AI 应用侧能力暴露。

MCP stdio HTTP JSON-RPC 工具注册

构建与接入路径

支持 CMake 工程化接入，提供 include 与 C++23 import/module 双路径，便于不同工具链下平滑落地。

CMake include import module examples tests
ecosystem

项目生态

Galay 不是单体框架，而是可组合的 C++23 库集合。能力覆盖运行时、协议服务、数据客户端与工具扩展，支持客户端和服务端双向场景。

C++23 模块编译支持

galay-* 系列库均提供命名模块接口，支持 import galay.xxx;。可按工具链选择 module/import 或传统 include，两条路径保持一致 API。

import galay.kernel;
import galay.http;
import galay.mysql;
import galay.mcp;
