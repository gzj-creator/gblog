# AI Service 接口参考

Base URL: `http://<ai-host>:8000`

## GET /

返回服务状态。

响应示例：

```json
{
  "status": "ok",
  "service": "Galay AI Service",
  "version": "2.1.0",
  "startup_error": null
}
```

## GET /health

返回健康检查信息。

响应示例：

```json
{
  "status": "ok",
  "startup_error": null,
  "services": {
    "vector_store_initialized": true,
    "chat_service_initialized": true,
    "index_state_watch_enabled": true,
    "index_state_watch_running": true
  },
  "index_version": 12
}
```

## POST /api/chat

请求：

```json
{
  "message": "galay-http 怎么快速开始？",
  "session_id": "default",
  "use_memory": true
}
```

响应：

```json
{
  "success": true,
  "response": "...",
  "sources": [
    {
      "project": "galay-http",
      "file": "README.md",
      "file_name": "README.md"
    }
  ],
  "blocks": [],
  "session_id": "default"
}
```

## POST /api/chat/stream

SSE 流式聊天接口。

请求体与 `/api/chat` 相同。

响应为 `text/event-stream`，事件数据格式：

```json
{"content":"..."}
{"done":true,"sources":[...]}
```

## POST /api/search

请求：

```json
{
  "query": "galay-mysql AsyncMysqlClient",
  "k": 3
}
```

响应：

```json
{
  "success": true,
  "results": [
    {
      "content": "...",
      "metadata": {
        "project": "galay-mysql",
        "source": "README.md"
      },
      "score": 0.92
    }
  ]
}
```

## 错误码

- `400` 参数错误
- `429` 请求限流
- `500` 服务内部错误
- `503` 向量索引不可用（未初始化或未就绪）
- `504` 大模型请求超时
